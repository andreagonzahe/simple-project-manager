# Inbox Quick Reference

## What is the Inbox?

A frictionless capture system for quick brain dumps. Add tasks without categories, organize them later.

## Quick Start

### Capture Anywhere (Dashboard)
1. Find "Quick Capture" card on dashboard
2. Type your thought
3. Press **Enter**
4. Done!

### Organize Later
1. Go to **Inbox** page (sidebar or mobile menu)
2. Click **folder icon** on any item
3. Choose area & project
4. Click **Organize**

## Locations

- **Dashboard**: Quick Capture card (fast entry)
- **Sidebar**: Inbox link (second item)
- **Mobile Menu**: Inbox link (at top)
- **Direct URL**: `/inbox`

## Key Actions

| Action | Where | How |
|--------|-------|-----|
| Quick Add | Dashboard card | Type + Enter |
| View All | Inbox page | Click "View Inbox" or sidebar link |
| Organize | Inbox page | Click folder icon on item |
| Delete | Inbox page | Click trash icon on item |
